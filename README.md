# Auto Organizer 
---
This is a project tht organizes the Downloads folder into seperated folders within it. 
Here, it organizes the following: 
- Archives
- Documents
- Installers
- Music 
- Misc
- Pictures
- Source Codes 
- Videos
- Zips

This makes the Downloads folder easier to navigate and traverse through, not having to look at heaps of unorganized files.
